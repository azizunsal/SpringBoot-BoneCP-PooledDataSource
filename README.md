## A SpringBoot Application with BoneCP
### Purpose

This example project demonstrates how to configure a [BoneCP](https://github.com/wwadge/bonecp) connection pool in a Spring Boot application. It was created to provide a concrete, working example in response to a Stack Overflow [question](https://stackoverflow.com/questions/28664064/spring-boot-not-a-managed-type).

### How To Run the Application
`mvn clean package && java -jar target/springbootbonecpds.jar`
